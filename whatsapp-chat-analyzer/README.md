# End-to-End WhatsApp Chat Analyzer

A full-stack data analytics dashboard built with Python and Streamlit. This application ingests raw exported WhatsApp `.txt` files, cleans the unstructured text using custom Regular Expressions, and performs exploratory data analysis (EDA) to extract actionable insights.

## 🛠️ Tech Stack
* **Frontend:** Streamlit
* **Data Processing:** Pandas, Regex (re)
* **Visualization:** Matplotlib, Seaborn
